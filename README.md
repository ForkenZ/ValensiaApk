# ⚗️ Valensia

**Aplikasi Android untuk Pembelajaran Kimia — dari SMA hingga Perkuliahan**

*Belajar kimia lebih mudah, interaktif, dan menyenangkan.*

</div>

---

## 📖 Tentang Valensia

**Valensia** adalah aplikasi Android yang dirancang untuk membantu siswa SMA dan mahasiswa dalam memahami kimia secara mendalam. Mulai dari materi dasar hingga konsep lanjutan, semua tersedia dalam satu aplikasi.

---

## ✨ Fitur Utama

| Fitur | Deskripsi |
|---|---|
| 🤖 **AI Valensia** | Tanya apa saja seputar kimia, dijawab langsung oleh AI |
| 📚 **Pembelajaran** | Materi lengkap dari SMA sampai tingkat kuliah |
| 🧪 **Lab Simulasi** | Simulasi eksperimen kimia tanpa perlu alat fisik |
| 🔬 **Tabel Periodik** | Tabel periodik interaktif dengan info lengkap tiap unsur |
| 📝 **Kuis** | Uji pemahaman dengan kuis berbasis materi |
| 🎮 **Games** | Belajar sambil bermain |
| 🏆 **Level Akun** | Sistem leveling untuk memotivasi belajar |

---

## 🤖 AI Valensia

AI bawaan aplikasi yang siap menjawab pertanyaan kimia kapan saja.

- Tanya soal rumus, reaksi, atau konsep kimia
- Penjelasan dalam bahasa yang mudah dipahami
- Cocok untuk belajar mandiri maupun persiapan ujian

---

## 🧪 Lab Simulasi

Lakukan eksperimen kimia secara virtual tanpa risiko.

- Simulasi reaksi kimia secara real-time
- Aman, gratis, dan bisa diulang berkali-kali
- Cocok untuk praktikum mandiri

---

## 🏆 Sistem Level Akun

Makin rajin belajar, makin tinggi level kamu!

- Kumpulkan XP dari kuis dan pembelajaran
- Unlock fitur dan konten eksklusif
- Pantau progress belajarmu

---

## 📱 Persyaratan

- Android **5.0 (Lollipop)** atau lebih tinggi
- Koneksi internet (untuk fitur AI)
- Storage minimal **50 MB**

---

## 🚀 Cara Install

1. Download file `.apk` dari halaman [Releases](../../releases)
2. Aktifkan **Install dari sumber tidak dikenal** di pengaturan Android
3. Install dan buka aplikasi
4. Daftar akun dan mulai belajar!

---

## 🛠️ Tech Stack

- **Platform** — Android (Java/Kotlin)
- **AI** — Anthropic Claude API / OpenAI API
- **Database** — Firebase / Room Database
- **UI** — Material Design 3
